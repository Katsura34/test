📚 JavaScript Methods & Properties

charCodeAt() - Get ASCII Code of Character

string.charCodeAt(index)
Purpose: Returns the ASCII (numeric) code of the character at a specific position
Parameters: index = position (0 = first character)
"A".charCodeAt(0) → 65
"B".charCodeAt(0) → 66
"hello".charCodeAt(0) → 104 (for 'h')
"hello".charCodeAt(1) → 101 (for 'e')
Used in code: char.toUpperCase().charCodeAt(0) - 65
Converts A-Z to 0-25 (A=0, B=1, ... Z=25)

String.fromCharCode() - Convert ASCII to Character

String.fromCharCode(code1, code2, ...)
Purpose: Converts ASCII numbers back into characters (opposite of charCodeAt)
String.fromCharCode(65) → "A"
String.fromCharCode(68) → "D"
String.fromCharCode(65, 66, 67) → "ABC"
Used in code: String.fromCharCode(charCode + 65)
Converts 0-25 back to A-Z letters

toLowerCase() - Convert to Lowercase

string.toLowerCase()
Purpose: Converts all uppercase letters to lowercase
"HELLO".toLowerCase() → "hello"
"Hello World".toLowerCase() → "hello world"
Used in code: To preserve original case after encryption

toUpperCase() - Convert to Uppercase

string.toUpperCase()
Purpose: Converts all lowercase letters to uppercase
"hello".toUpperCase() → "HELLO"
"Hello World".toUpperCase() → "HELLO WORLD"
Used in code: text.toUpperCase() before encryption (Caesar & Hill use A-Z only)

slice() - Extract Part of String

string.slice(start, end)
Purpose: Extracts a section of a string
Parameters:
• start = starting position
• end = ending position (NOT included)
• Negative numbers count from end: -1 = last char, -2 = second to last
"hello".slice(0, 2) → "he"
"hello".slice(-1) → "o" (last character)
"hello".slice(-2) → "lo" (last 2 characters)
"hello".slice(2) → "llo" (from position 2 to end)
Used in code: result.slice(0, -1) + result.slice(-1)
Converts last character to lowercase while keeping rest uppercase

.length - Get String Length

string.length
Purpose: Returns the number of characters (NOT a function, it's a property)
"hello".length → 5
"a".length → 1
"".length → 0
Used in code: plaintext.length % 2 !== 0
Checks if text length is odd (for Hill Cipher pair processing)

match() - Check if String Matches Pattern

string.match(/pattern/flags)
Purpose: Checks if string matches a regular expression
/[a-z]/i means: letters a-z, i = case-insensitive
"A".match(/[a-z]/i) → matches
"5".match(/[a-z]/i) → null (doesn't match)
"!".match(/[a-z]/i) → null
Used in code: if (char.match(/[a-z]/i))
Only encrypt letters, leave numbers and special chars unchanged

=== Strict Equality

value1 === value2
Purpose: Checks if two values are exactly equal (same value AND same type)
5 === 5 → true
"5" === 5 → false (different types)
"caesar" === "caesar" → true
Used in code: cipher === "caesar"
Checks if selected cipher is Caesar type

% Modulo - Get Remainder

number1 % number2
Purpose: Returns the remainder after division
10 % 3 → 1
28 % 26 → 2
5 % 2 → 1 (5 is odd)
4 % 2 → 0 (4 is even)
Used in code:
• (charCode + key + 260) % 26 → wraps to 0-25
• length % 2 !== 0 → checks if odd

? : Ternary Operator

condition ? valueIfTrue : valueIfFalse
Purpose: Quick if/else in one line
operation === "encrypt" ? "Encrypted" : "Decrypted"
Used in code: const key = operation === "encrypt" ? shift : -shift;
If encrypting, use +3. If decrypting, use -3.

• Loop runs 1 time (length 2)
• i = 0: process "HI" (chars at positions 0 and 1)

For "HELLO" (becomes "HELLOX"):
• Loop runs 3 times (length 6)
• i = 0: process "HE"
• i = 2: process "LL"

For "HI":

• I.charCodeAt(0) = 73 → 73 - 65 = 8 (p2 = 8)

P = [[7],
     [8]]

11 let c2 = (key[1][0] * p1 + key[1][1] * p2) % 26;
Matrix multiplication formula:
| 2 1 | | 7 |
| 5 1 | × | 8 |

Step 12:

Step 13:
c2 = 17 → 17 + 65 = 82 → String.fromCharCode(82) = "R"

🔓 Hill Cipher Decryption (Complex!)

To decrypt, we need the INVERSE of the key matrix. The process involves:
10 CALCULATE DETERMINANT
1 let det = (key[0][0] * key[1][1]) - (key[0][1] * key[1][0]);
For key [[2,1], [5,1]]:
det = (2×1) - (1×5) = 2 - 5 = -3
In modulo 26: -3 ≡ 23 (mod 26)
11 FIND MODULAR INVERSE OF DETERMINANT
2 let invDet = 0;
3 for (let i = 1; i < 26; i++) {
4     if ((det * i) % 26 === 1) {
Find number where (det × i) % 26 = 1
(23 × i) % 26 = 1
Testing: 23×17 = 391 = 15×26 + 1 ✓
So invDet = 17
